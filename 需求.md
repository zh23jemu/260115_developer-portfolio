# 个人作品集网站 PRD

## 核心功能
1. 首页：大标题 + 简介 + 头像
2. 关于我：详细介绍 + 技能列表
3. 项目展示：项目卡片列表，每个卡片包含项目名称、项目截图、简短描述、技术栈、项目链接
4. 联系方式：邮箱、GitHub、社交媒体链接

## 设计要求
- 简洁现代的设计风格
- 深色主题
- 流畅的滚动动画
- 移动端适配

## 设计要求
- 深色主题（背景 #0a0a0a，文字 #ffffff）
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端适配

## 注意事项
- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
- 确保所有链接可点击
